## Question {nump}`question`{points}`3`

Calculez:

{.lower-alpha-paren .vsep-2}
1.  $64^{-\frac{1}{3}} = $ {leader}`.`
2.  $\log(10) = $ {leader}`.`
3.  $\log_3(27) = $ {leader}`.`
4.  $\log_2(\sqrt{2}) = $ {leader}`.`
5.  $\log(\dfrac{1}{1000}) = $ {leader}`.`
6.  $4^{\log_4(5)} = $ {leader}`.`



```{solution}
{.lower-alpha-paren .vsep-2}
1.  $64^{-\frac{1}{3}} = \dfrac{1}{4}$
2.  $\log(10) = 1$
3.  $\log_3(27) = 3$
4.  $\log_2(\sqrt{2}) = \dfrac{1}{2}$
5.  $\log(\dfrac{1}{1000}) = -3$
6.  $4^{\log_4(5)} = 5$
```

## Question {nump}`question`{points}`3`

Que vaut x ?

{.lower-alpha-paren .vsep-2}
1.  $5^x = 125 \qquad x = $ {leader}`.`
2.  $4^x = \dfrac{1}{16} \qquad x = $ {leader}`.`
3.  $7^{3x} = 7^{6} \qquad x = $ {leader}`.`
4.  $\log(x) = 4 \qquad x = $ {leader}`.`
5.  $\log_x(49) = 2 \qquad x = $ {leader}`.`
6.  $\log_3(x) = 0 \qquad x = $ {leader}`.`



```{solution}
{.lower-alpha-paren .vsep-2}
1.  $5^x = 125 \qquad x = 3$
2.  $4^x = \dfrac{1}{16} \qquad x = -2$
3.  $7^{3x} = 7^{6} \qquad x = 2$
4.  $\log(x) = 4 \qquad x = 10\,000$
5.  $\log_x(49) = 2 \qquad x = 7$
6.  $\log_3(x) = 0 \qquad x = 1$
```
